FIXM Flight Plan Examples

The core focus of FIXM version 1 has been to model the data elements in ICAO format flight plans (FPL). This page presents some example ICAO flights plans and the corresponding FIXM XML document. The FIXM documents have been generated by an Airservices Australia utility that reads ICAO format flight plans, decodes them, and generates the corresponding FIXM objects. The flight plan examples are operational flight plans that have passed through Airservices Australia's AFTN message switch.

From a technical perspective:

  • Java classes are generated by JAXB from the FIXM schema;
  • The flight plan is parsed and decoded, and the JAXB classes used to create a Java object corresponding to the flight plan;
  • JAXB is used to marshal the Java object and create the FIXM XML document.

For further information on the translation between ICAO flight plans and FIXM documents, refer to FIXM via FPL.

The example ICAO flight plans are accessed via the links below. Each example consists of a FIXM XML document, with the ICAO FPL from which it was derived displayed as a comment at the top of the XML document. The examples comply with the FIXM V1.1 schema.
